# 📱 Flutter UI Clone — Figma to Flutter

This project is a **UI cloning implementation** built with Flutter, based on a design from Figma.

The main goal of this project is to demonstrate how to convert a design into a clean, structured, and pixel-perfect Flutter UI — **without any backend or business logic**.

---

## ✨ Features

- 🎯 Pixel-perfect UI implementation
- 🧱 Clean and structured layout
- 📐 Based on real Figma design
- ⚡ Lightweight (UI only, no API, no state management)

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Flutter SDK
- Dart
- Android Studio / VS Code

### Installation

```bash
git clone https://github.com/your-username/flutter-ui-clone.git
cd flutter-ui-clone
flutter pub get
flutter run
```
